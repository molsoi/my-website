        # my-site

This is a ready-to-deploy React + Vite + Tailwind project.

## Quick commands

1. Install dependencies:

```
npm install
```

2. Run dev server:

```
npm run dev
```

3. Build for production:

```
npm run build
```

4. Deploy with Vercel (recommended):

```
vercel --prod
```

Or drag the `dist` folder to Netlify Drop.
